# Snake & Ladder Game

## Project Description
This is a 2-player terminal-based Snake & Ladder game written in Python. Players take turns rolling a dice and moving across a board from 1 to 100. The game includes snakes, ladders, replay functionality, a scoreboard, and a visual board display.

## Features
- 2-player gameplay
- Dice roll simulation using 
- Snakes and ladders stored using dictionaries
- Exact landing on 100 required to win
- Player name input
- Replay option
- Scoreboard to track wins
- Visual 10x10 board display
- Colored player symbols on the board

## Data Structures Used
- **Dictionaries**
  - snake positions
  - ladder positions
  - scoreboard tracking
- **Lists**
  - used to construct the visual board
